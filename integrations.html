How to Integrate

With Safely Finance you have options for how you'd like to integrate. These aren't mutually exclusive. You can opt to use Safely Checkout alone, or, use Safely Checkout in addition to the Safely API integration. You can read up more on both options below.

Safely Checkout​

A simple approach that doesn't require calls to the Safely Finance API. Safely Checkout via safely.js is the quickest option to get Safely Finance up and running on your website.

Safely API (enhanced integration)​

A more integrated approach to implementing Safely Finance on your website. Use of the API gives you flexibility to check on the status of a checkout. As well as light analytics on how your users are interacting with the Safely Checkout experience.
